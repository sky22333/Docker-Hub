使用说明

  1. 支持GitHub文件、Releases、archive、raw文件下载
  2. 输入GitHub文件的完整链接，然后点击下载，或者复制。
  3. 输入的GitHub链接需要带https，支持git clone等终端命令

终端命令行示例

git clone

git clone https://gh-proxy.com/https://github.com/containers/podman.git

wget

wget https://gh-proxy.com/https://github.com/containers/podman/archive/refs/tags/v5.6.0.zip

curl

curl -LO https://gh-proxy.com/https://github.com/containers/podman/archive/refs/tags/v5.6.0.zip
